Виджеты 

Выручка
- Источник: ПФ 
- Прошлые месяцы: сумма  по статьям с типом Income
- Будущие: из бюджета БДР
- Культ: исключены проекты Тех. отдел, СнупДок, Бурлеск

Маржа
- Источник: ПФ
- Формула: Выручка − Переменные расходы

Марж-ть
- Источник: ПФ
- Формула: (Маржа / Выручка) × 100%

Прибыль / Убыток
- Источник: ПФ 
- Прошлые месяцы: Доходы − Расходы
- Будущие: из бюджета

На счетах
- Источник: ПФ
- Текущий остаток на всех счетах на сегодня

Проектов
- Источник: АМО
- Бластер: фильтр статус "Продано" + дата акта в периоде
- Культ: фильтр этапы препродакшн, постпродакшн, ждём оплату , фин. и док. отчёт , реализовано + дата акта в периоде


График: Чистая прибыль и рентабельность

- Источник: ПФ
- Левая ось (деньги):
  - Бюджет НИ — накопительная сумма  по месяцам
  - Факт НИ — накопительная сумма  (только прошлые месяцы)
- Правая ось (%):
  - Рентабельность — (накопФактПрибыль / накопФактВыручка) × 100%
  - Только для прошлых месяцев


График: Бизнес-уравнение

Показывает отклонение факта от бюджета в процентах.

Бластер (9 столбиков):

- Запросы — факт: AMO, все лиды в воронке по дате создания (все этапы); бюджет: захардкожен [7,19,22,11,14,20,20,30,30,32,16,0]
- Конверсия — факт: sold / (sold + notSold) × 100%; бюджет: 50%
- Проекты — факт: кол-во проектов по актам из AMO; бюджет: выручка из бюджета / 647 500
- Средний чек — факт: выручка / кол-во проектов по актам; бюджет: 647 500
- Выручка — факт: ПФ; бюджет: ПФ бюджет БДР
- Маржин-ть — факт: средний % маржинальности за прошлые месяцы; бюджет: средний % из бюджета
- Маржа — факт: ПФ; бюджет: ПФ
- Пост. расходы — факт: ПФ; бюджет: ПФ
- Прибыль — факт: ПФ; бюджет: ПФ

Культ (5 столбиков):
- Выручка, Маржин-ть, Маржа, Пост. расходы, Прибыль
- Запросы, Конверсия, Проекты, Средний чек — убраны (пока не ясно откуда брать)

Формула отклонения:
- `((факт − бюджет) / |бюджет|) × 100%`
- Ограничено [-100%, 100%] на графике


График: Маржинальность

- Источник: АМО (проекты с ценой и планом расходов/маржой)
- НИ (первый столбик) — средняя маржинальность по всем проектам за период
- Помесячные столбики — маржинальность проектов за каждый месяц
- Клик на месяц — детализация по проектам
- Пунктирная линия — средний budgetMarginPercent из ПФ


График: Исполнение бюджета расходов

- Факт: ПФ (кумулятивный за прошлые месяцы)
- Бюджет: ПФ бюджет БДР (за весь период, включая будущие месяцы)
- Группировка по статьям 1-го уровня под корневой категорией "Расходы"

Фильтр статей:

Бластер:
- ЗП Бэкофис, ЗП Продакшн, ЗП Коммерция, ЗП КРЕАТИВ
- ОФИС, КОМИССИИ, НАЛОГИ ФОТ
- БОНУСЫ НЕ В МАРЖЕ, ИНВЕСТИЦИИ/СТРАТЕГИЧЕСКИЕ АКТИВНОСТИ, ИМИДЖ И РАЗВИТИЕ

Культ:
- ЛЮДИ, БЭКОФИС, ОФИС
- Налог на прибыль (доходы), Налоги и комиссии
- ФИН. ОПЕРАЦИИ, ТЕНДЕРЫ, Проценты по кредитам и займам
- ИМИДЖ, Онлайн сервисы, БОНУСЫ НЕ В МАРЖЕ

Столбики:
- Синий — факт (в пределах бюджета)
- Розовый — перерасход (факт > бюджет)
- Мятный — экономия (факт < бюджет)
- Подпись сверху — % отклонения
